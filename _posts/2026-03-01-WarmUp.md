## Information (Forensic - Easy)

Pertama Kita akan Download File yang di sediakan PicoCTF menggunakan Command Wget Di Linux

![Download File](/content/img/warmupidn/download.png)

Selanjutnya sebelum saya menelusuri file lebih jauh, Hal yang pertama kali saya lakukan adalah melihat Metadata dari file tersebut menggunakan Tools Exiftool

![Exiftool](/content/img/warmupidn/exiftool.png)

Dari Metadata tersebut, Ada banyak informasi yang berguna dalam Forensic seperti Tipe File,Kapan File di modifikasi,Ukuran Dan Lain sebagainya.

Disini saya melihat keanehan pada bagian Licence yang berisi kata Acak atau encoding dengan algoritma tertentu.

Setelah saya searching di internet, Kata tersebut di encode menggunakan algoritma base64, Lalu saya menggunakan command Bawaan dari kali linux untuk mengdecode kalimat tersebut
